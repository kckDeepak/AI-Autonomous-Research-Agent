# P0-S2 Source Acceptance Criteria

## Goal

Define which sources are accepted into synthesis and which are filtered.

## Inclusion Rules

1. URL reachable and content extractable.
2. Content is materially related to the query/subtopic.
3. Source has identifiable publisher/author/organization metadata when available.
4. Content has enough substance (not just navigation pages or thin snippets).

## Exclusion Rules

1. Duplicate or near-duplicate URLs/content.
2. Link farms, obvious spam, or scraped aggregators with no primary value.
3. Pages dominated by ads or non-content boilerplate.
4. Access-restricted pages where meaningful text cannot be extracted.

## Quality Heuristics

- Prioritize primary and reputable secondary sources.
- Prefer recent publication dates for fast-moving topics.
- Enforce domain diversity to reduce monoculture bias.
- Record rejection reason for observability.

## Minimum Bar (Standard Depth)

- Target 10-20 accepted sources.
- Minimum accepted source count: 10 (or explicit degraded status if lower).
