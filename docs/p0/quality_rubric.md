# P0-S2 Report Quality Rubric

## Scoring Dimensions (0-5 each)

| Dimension         | 0-1                    | 2-3                             | 4-5                                          |
| ----------------- | ---------------------- | ------------------------------- | -------------------------------------------- |
| Relevance         | Mostly off-topic       | Partially aligned               | Directly answers the query                   |
| Grounding         | Claims lack sources    | Some claims sourced             | Claims consistently source-backed            |
| Citation Coverage | Sparse/unclear links   | Links present but inconsistent  | Every key finding has traceable source(s)    |
| Source Diversity  | Narrow source set      | Mixed but clustered domains     | Balanced across credible, diverse domains    |
| Synthesis Quality | List of summaries only | Basic cross-source synthesis    | Clear synthesis, contrasts, and implications |
| Readability       | Hard to follow         | Understandable with rough edges | Clear structure, concise, executive-ready    |

## Acceptance Threshold

- Minimum average score: `>= 3.5 / 5`
- Hard minimum for grounding and citation coverage: `>= 4 / 5`

## Required Report Sections

1. TL;DR
2. Executive Summary
3. Key Findings
4. Deep Dives (Top 3)
5. References (URL list)

## Automatic Rejection Triggers

- Any major finding without at least one citation
- References missing URL or unreachable at extraction time
- Fabricated or non-attributable statistics
